# Adverse Events SDTMv Dataset

An example Adverse Events (AE) SDTMv domain.

## Usage

``` r
datacutr_ae
```

## Format

A dataset with 5 rows and 3 variables:

- USUBJID:

  Unique Subject Identifier

- AETERM:

  Reported Term for the Adverse Event

- AESTDTC:

  Start Date/Time of Adverse Event
